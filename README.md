# 📊 Tableau de Bord de Suivi d'Activité – Santé au Travail

> Application web interactive développée avec **Python + Streamlit + Plotly** pour le pilotage en temps réel de l'activité d'un service de santé au travail. Visualisation des KPI, tendances et indicateurs de performance.

## 🚀 Démo en ligne

👉 **[[Accéder au tableau de bord]([LIEN_STREAMLIT_ICI](https://activity-dashboard-k5fwshrk7wulatry4j9jri.streamlit.app/))](https://activity-dashboard-k5fwshrk7wulatry4j9jri.streamlit.app/)**

---

## 🎯 Objectif

Centraliser et visualiser les données d'activité médicale pour permettre aux équipes de direction de **piloter l'activité en temps réel** et d'identifier les leviers d'amélioration.

---

## 📊 KPI et visualisations

| Indicateur | Description |
|---|---|
| **Taux de réalisation** | Visites réalisées / programmées |
| **Adhérents actifs** | Nombre et répartition sectorielle |
| **Score d'engagement** | Indice par secteur d'activité |
| **Risques identifiés** | Top 5 des risques professionnels |
| **Visites urgentes** | Suivi mensuel des urgences |

---

## 🛠️ Stack technique

| Outil | Rôle |
|---|---|
| Python 3.8+ | Langage principal |
| Streamlit | Interface web interactive |
| Plotly | Graphiques interactifs |
| Pandas / NumPy | Traitement des données |
| Power BI | Tableaux de bord métier (version entreprise) |

---

## 🚀 Installation & Lancement

```bash
git clone https://github.com/KenewyD/activity-dashboard.git
cd activity-dashboard
pip install -r requirements.txt
streamlit run app.py
```

---

## ⚙️ Fonctionnalités

- 🎛️ **Filtres interactifs** : période et secteurs
- 📈 **6 visualisations** : barres, courbes, camembert, histogramme
- 🎯 **Seuils d'alerte** : ligne objectif 90% sur taux de réalisation
- 📱 **Responsive** : adapté desktop et tablette
